# gulp-template

Minimal template for building HTML pages with [gulp](https://gulpjs.com/).

## Installing

```bash
git clone https://github.com/nikolaas/gulp-template.git my-project
cd my-project
rm -rf .git
git init
```

## Usage

Build a production ready build:

```bash
npm run build
```

Run development server:

```bash
npm start
```
